# GroupDocs.Viewer — landing page

GitHub Pages site for **GroupDocs.Viewer**, served at <https://groupdocs-viewer.github.io/>.

## Maintenance
- **`index.html` is generated — do not edit it by hand.** It is produced by the
  GroupDocs landing-page generator from a shared design plus live product data
  (versions, GitHub stars, and download counts from
  <https://products.groupdocs.com/versions.json> — baked in at build time and
  refreshed in the browser on load).
- `groupdocs.svg` and `favicon.ico` are assets used by the page.

To change the page, update the generator inputs and re-run it — don't hand-edit `index.html`.
